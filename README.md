lighthouse-role
=========

Эта роль производит установку Lighthouse

Role Variables
-------------
|  Variable  |    |
|:---|:---|
| lighthouse_dir | Директория для загрузки файлов из репозитория Lighthouse |

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: lighthouse }
